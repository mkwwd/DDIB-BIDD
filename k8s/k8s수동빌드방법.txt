**삭제는 조심 조심 또조심**

 docker build -t kimyusan/k8s_product:latest .


 
 docker push kimyusan/k8s_product:latest


 k8s master 서버 접속

kubectl get pod -n ddib ( ddib 의 파드를 조회합니다.)


kubectl delete pod ddib-product-pod -n ddib(ddib 에서 ddib-product-pod 를 삭제합니다)

cd ~/ddib/product/ (파드 설정파일있는 위치로 들어갑니다.)

kubectl apply -f ddib-product-pod.yaml  (ddib-product-pod 의 설정이담겨있는 yaml 파일을 적용합니다.) 


kubectl get pod -n ddib ( ddib 의 파드를 조회합니다.)

kubectl logs ddib-product-pod -n ddib (해당 파드의 로그를 조회합니다.)
